# Term Dash
Geometry dash in your terminal.

Written in Bevy and Rust. Uses `bevy_ratatui` and `bevy_ratatui_camera` to render in the terminal.

I've been wanting to make a game for a while and learn Rust and Bevy. The terminal is cool. Geometry dash is a classic game. Combine them :)

Made for Hack Club Horizons!

**Work in progress**

# Usage
As of now, there are no precompiled builds. You can clone the repository:

```bash
git clone https://github.com/intercepted16/termdash
```

and then use:

```bash
cargo run
```

to run it, which will handle everything for you; though note that it hasn't been tested on non-Linux systems, however, Bevy (and obviously Rust) are cross-platform.
